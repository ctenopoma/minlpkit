# Sample Catalog

This is a catalog listing **131** MINLP/MILP sample models bundled with minlpkit by category (equivalent to the scikit-learn Example gallery).
Each sample has a `build_model()` based on a real business problem, and can be used as a testbed for visualization, diagnosis, and improvement methods. To see the execution results, refer to the [Outcome Gallery](../gallery.en.md).
The description is automatically extracted from the module docstring of each file (Regeneration: `uv run python experiments/gen_sample_catalog.py`).

The ⭐ mark indicates samples with particularly detailed business stories (11 models). Some of them come with a learning notebook (`docs/notebooks/samples/`) that allows you to execute the story of business story -> naive formulation -> diagnosis -> improvement as a single narrative (those with a "notebook" link in the table below apply).
