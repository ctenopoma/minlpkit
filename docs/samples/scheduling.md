# スケジューリング

ジョブショップ・シフト・保守・列車運行など時間軸の割当計画。本リポジトリ最大のカテゴリで、対称性や時間結合の題材が揃う。

**21 本** / `scale` 引数対応 1 本。 ⭐ は事業ストーリーが特に厚い旗艦サンプル。`scale` 列 ✓ は `build_model(scale=...)` で規模可変。
