MINLP 可視化・診断・改善

PySCIPOpt(SCIP)によるMINLPの求解プロセスを可視化し、診断ルールで改善を提案し、 効果を検証する一連の成果。統合API minlpkit / 一気通貫デモ demo.py。
ライブモニタ(要サーバ起動): uv run python -m viz.server の後 http://127.0.0.1:5000/ を開く。2 run の重ね描画による比較モードあり。
